# Винлокер (Демонстрационный проект)

Это демонстрационный проект, имитирующий работу вредоносного ПО типа "винлокер" (winlocker) в образовательных целях. Проект создан для демонстрации техник социальной инженерии и методов, используемых злоумышленниками.

## ⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ

**Данный проект создан ИСКЛЮЧИТЕЛЬНО в образовательных целях!**

* Это НЕ настоящее вредоносное ПО
* НЕ используйте этот код для незаконных действий
* Проект демонстрирует методы социальной инженерии для повышения осведомленности о кибербезопасности

## Особенности проекта

* Имитация блокировки экрана компьютера
* Таймер обратного отсчета для создания ощущения срочности
* Имитация активации веб-камеры
* Форма для ввода платежных данных (не отправляет никакие данные)
* Визуальные и звуковые эффекты для создания тревожной атмосферы
* Информационное модальное окно, объясняющее, что это демонстрация

## Технологии

* HTML5
* CSS3 (анимации, адаптивный дизайн)
* JavaScript (без использования внешних библиотек)

## Как запустить

1. Откройте файл `index.html` в любом современном браузере
2. Для выхода из демонстрации используйте кнопку "Понятно, закрыть демонстрацию" в модальном окне

## Образовательная ценность

Данный проект помогает понять:

* Как работают программы-вымогатели
* Какие психологические приемы используют мошенники
* Как распознавать фишинговые атаки
* Почему важно не паниковать при столкновении с подозрительными сообщениями

## Правила безопасности

Если вы столкнулись с настоящим винлокером:

1. Не паникуйте
2. Не платите выкуп
3. Перезагрузите компьютер в безопасном режиме
4. Используйте антивирусное ПО для удаления вредоносных программ
5. Обратитесь к специалистам по информационной безопасности
