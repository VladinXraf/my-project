# GameKeys - Сайт с готовыми ключами от платных игр

Современный веб-сайт для получения готовых ключей активации от популярных платных игр на различных игровых платформах.

## 🎮 Особенности

- **Готовые ключи** от реальных платных игр (не генерируются случайно!)
- **Красивый современный дизайн** с градиентами и анимациями
- **Поддержка популярных платформ**: Steam, Epic Games, Origin, Ubisoft Connect
- **Фильтрация игр** по платформам
- **Поиск игр** по названию и описанию
- **Модальные окна** с готовыми ключами и инструкциями по активации
- **Адаптивный дизайн** для всех устройств
- **Плавные анимации** и переходы
- **Интерактивные элементы** с hover-эффектами

## 🚀 Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с градиентами, анимациями и backdrop-filter
- **JavaScript (ES6+)** - интерактивность и функциональность
- **Font Awesome** - иконки
- **Google Fonts** - шрифт Inter

## 📱 Платформы

- **Steam** - 250+ игр
- **Epic Games** - 120+ игр  
- **Origin** - 80+ игр
- **Ubisoft Connect** - 60+ игр

## 🚀 Запуск

1. Откройте файл `index.html` в любом современном браузере
2. Или запустите локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```

## 🔧 Настройка

### Добавление новых игр с готовыми ключами:
1. Откройте `index.html`
2. Найдите секцию `#games .games-grid`
3. Добавьте новую карточку игры по образцу существующих:
   ```html
   <div class="game-card" data-platform="steam" data-price="49.99">
       <div class="game-image">
           <img src="путь_к_изображению.jpg" alt="Название игры">
           <div class="platform-badge steam">
               <i class="fab fa-steam"></i>
           </div>
       </div>
       <div class="game-info">
           <h3>Название игры</h3>
           <p>Описание игры</p>
           <div class="game-meta">
               <span class="original-price">49.99$</span>
               <span class="free-badge">БЕСПЛАТНО</span>
           </div>
           <button class="get-key-btn" data-game="Название игры" data-key="ГОТОВЫЙ-КЛЮЧ-ЗДЕСЬ">
               <i class="fas fa-key"></i>
               Получить ключ
           </button>
       </div>
   </div>
   ```

### Изменение цветов:
1. Откройте `styles.css`
2. Найдите CSS-переменные в начале файла
3. Измените значения градиентов и цветов

### Добавление новых платформ:
1. Добавьте иконку платформы в секцию платформ
2. Добавьте соответствующие стили в CSS
3. Обновите JavaScript фильтрацию
