# CodeWorld - Сайт о программировании

Современный веб-сайт для изучения программирования с красивым дизайном и интерактивными элементами.

## 🚀 Особенности

- **Современный дизайн** - Красивый и отзывчивый интерфейс
- **Адаптивная верстка** - Работает на всех устройствах
- **Интерактивные элементы** - Анимации и эффекты при прокрутке
- **Плавная навигация** - Удобное перемещение по разделам
- **Форма обратной связи** - С валидацией и уведомлениями

## 📱 Разделы сайта

1. **Главная** - Приветственная секция с анимированным кодом
2. **Языки программирования** - Обзор популярных языков
3. **Уроки** - Обучающие материалы по программированию
4. **Проекты** - Примеры практических проектов
5. **Контакты** - Форма для связи с командой

## 🛠️ Технологии

- **HTML5** - Семантическая разметка
- **CSS3** - Современные стили и анимации
- **JavaScript** - Интерактивность и функциональность
- **Font Awesome** - Иконки
- **Google Fonts** - Шрифт Inter

## 🎨 Дизайн

- Градиентные фоны
- Карточки с тенями и hover-эффектами
- Анимации появления элементов
- Плавные переходы и трансформации
- Цветовая схема в стиле IT

## 📱 Адаптивность

- Мобильное меню-гамбургер
- Адаптивные сетки
- Оптимизация для всех размеров экранов
- Touch-friendly интерфейс

## 🚀 Запуск

1. Откройте `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   python -m http.server 8000
   ```
3. Откройте `http://localhost:8000`
