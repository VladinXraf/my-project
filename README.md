# Академия Успеха - Платформа платных курсов

Современный веб-сайт для продажи онлайн-курсов с красивым дизайном и системой оплаты.

## 🚀 Особенности

- **Современный дизайн** - Красивый и адаптивный интерфейс
- **Платная система курсов** - Три курса с разными ценами
- **Система оплаты** - Модальное окно для ввода данных карты
- **Адаптивность** - Работает на всех устройствах
- **Анимации** - Плавные переходы и эффекты
- **Мобильное меню** - Удобная навигация на мобильных устройствах

## 🚀 Запуск проекта

1. Откройте файл `index.html` в браузере
2. Или запустите локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```
