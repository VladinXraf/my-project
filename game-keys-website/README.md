# GameKeys - Сайт с бесплатными ключами игр

Современный веб-сайт для получения бесплатных ключей игр на различных игровых платформах.

## 🎮 Особенности

- **Красивый современный дизайн** с градиентами и анимациями
- **Поддержка популярных платформ**: Steam, Epic Games, Origin, Ubisoft Connect
- **Фильтрация игр** по платформам
- **Поиск игр** по названию и описанию
- **Генерация ключей** с красивыми модальными окнами
- **Адаптивный дизайн** для всех устройств
- **Плавные анимации** и переходы
- **Интерактивные элементы** с hover-эффектами

## 🚀 Технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с градиентами, анимациями и backdrop-filter
- **JavaScript (ES6+)** - интерактивность и функциональность
- **Font Awesome** - иконки
- **Google Fonts** - шрифт Inter

## 📱 Платформы

- **Steam** - 150+ игр
- **Epic Games** - 80+ игр  
- **Origin** - 45+ игр
- **Ubisoft Connect** - 30+ игр

## 🚀 Запуск

1. Откройте файл `index.html` в любом современном браузере
2. Или запустите локальный сервер:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve .
   
   # PHP
   php -S localhost:8000
   ```

## 🔧 Настройка

### Добавление новых игр:
1. Откройте `index.html`
2. Найдите секцию `#games .games-grid`
3. Добавьте новую карточку игры по образцу существующих

### Изменение цветов:
1. Откройте `styles.css`
2. Найдите CSS-переменные в начале файла
3. Измените значения градиентов и цветов

### Добавление новых платформ:
1. Добавьте иконку платформы в секцию платформ
2. Добавьте соответствующие стили в CSS
3. Обновите JavaScript фильтрацию
